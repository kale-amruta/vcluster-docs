Once you have created a namespace template you can then refer to that template when
creating a new namespace. All template configurations will be automatically applied to the
newly created namespace. Namespaces referring to templates can be created in the UI
or via the vCluster CLI.

<Tabs
    defaultValue="ui"
    values={[
        {label: 'UI', value: 'ui'},
        {label: 'CLI', value: 'cli'},
    ]}>
    <TabItem value="ui">
        <Flow id="use-namespace-template-ui">
            <Step>
                Select the <NavStep>Projects</NavStep> field on the left menu bar.
            </Step>
            <Step>
                From the project drop-down menu, select the project you'd like to create the
                namespace in.
            </Step>
            <Step>
                Click the <NavStep>Namespaces</NavStep> option on the project pane.
            </Step>
            <Step>
                Click the <Button>Create Namespace</Button> button on the right side of
                the page.
            </Step>
            <Step>
                In the pop-up box, select your template from the template drop down menu.
            </Step>
            <Step>
                <b>[Optional]</b> select the cluster in which to create the virtual cluster.
            </Step>
            <Step>
                Click the <Button>Select</Button> to continue.
            </Step>
            <Step>
                In the drawer that appears from the right, give your namespace a name by
                replacing the <Label>my-namespace</Label> placeholder name, or by updating the
                manifest YAML <Label>metadata.name</Label> field.
            </Step>
            <Step>
                Click the <Button>Create Namespace</Button> button.
            </Step>
        </Flow>
    </TabItem>
    <TabItem value="cli">
        <PartialNamespaceTemplateInstantiateCLI/>
    </TabItem>
</Tabs>
