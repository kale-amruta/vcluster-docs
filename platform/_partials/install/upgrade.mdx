Upgrade vCluster Platform via:

<TabItem value="cli">

To upgrade vCluster Platform via vCluster CLI, update `$PLATFORM_VERSION` with a valid vCluster Platform version and run:

```bash
RELEASE_NAMESPACE=vcluster-platform # Replace with the namespace of the platform deployment if different
PLATFORM_VERSION='' # Set this to a specific version or leave empty for latest

vcluster platform start --upgrade --version=$PLATFORM_VERSION --namespace=$PLATFORM_NAMESPACE --values=vcluster-platform.yaml
```

</TabItem>
<TabItem value="helm">

To upgrade vCluster Platform via `helm`, update `$PLATFORM_VERSION` with a valid vCluster Platform version and run:

```bash
RELEASE_NAME=vcluster-platform # Replace with the release name if different
RELEASE_NAMESPACE=vcluster-platform # Replace with the namespace of the platform deployment if different
PLATFORM_VERSION='' # Set this to a specific version or leave empty for latest


helm upgrade $RELEASE_NAME vcluster-platform -n $PLATFORM_NAMESPACE --repository-config '' --repo https://charts.loft.sh \
  --version $PLATFORM_VERSION \
  --reuse-values \
  -f vcluster-platform.yaml
```

</TabItem>
