To get started creating a Project, in the vCluster Platform UI:

<Flow id="create-project-ui">
  <Step>
    All available projects are in the top left dropdown. Click the toggle to find the <Label>+ New Project</Label> option.
  </Step>
   <Step>
    Click on <NavStep>Members</NavStep> and add your users or teams and
    their project roles. See the <Link to="./using-projects/members">Manage Project Members</Link>{" "}
    section for more information.
  </Step>
  <Step>
    Click on <NavStep>Allowed Templates</NavStep> to set the templates
    allowed for the project. By default, templates are required and can be changed in this step. See{" "}
    <Link to="./using-projects/templates">Manage Allowed Templates</Link> for more information.
  </Step>
  <Step>
    Click on <NavStep>Allowed Clusters</NavStep> to set the clusters that
    members should have access to. See the <Link to="./using-projects/clusters">Manage Allowed Clusters</Link>{" "}
    section for further information.
  </Step>
  <Step>
    Click on the <NavStep>Configure Project Quotas</NavStep> to set the quotas for the project or specific user/team. See the <Link to="./using-projects/quotas">Quotas</Link>{" "}
    section for further information.
  </Step>
  <Step>
    Optionally, there are a couple of other integrations that can be configured. See
    the <Link to="../integrations/argocd">Integrations</Link> documentation for further information.
  </Step>
  <Step>
    Once all project options have been specified, click the{" "}
    <Button>Save Changes</Button>.
  </Step>
</Flow>
